## OrbitalMissile Class

Missile that will orbit the specified target\.

Note: this missile calls [OnImpact\(\)](WCSharp.Missiles.Missile.OnImpact().md 'WCSharp\.Missiles\.Missile\.OnImpact\(\)') when the target dies.
